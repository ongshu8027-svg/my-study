1. 클래스 기초 개념 정리
클래스와 객체의 정의
가장 흔히 쓰이는 비유는 '설계도와 건축물'입니다.

클래스 (Class): 객체를 만들기 위한 설계도나 틀입니다. 어떤 데이터가 들어가고 어떤 기능이 작동할지 정의만 해둔 상태이며, 실제 메모리 공간을 차지하지 않습니다.

객체 (Object): 설계도(클래스)를 바탕으로 실제 메모리에 구현된 실체(Instance)입니다.

클래스 구성 요소
클래스는 크게 두 가지의 핵심 요소로 이루어집니다. "무엇을 가지고 있고(상태), 무엇을 할 수 있는가(행동)"를 정의합니다.

필드 (Fields / 멤버 변수 / 상태):

객체의 데이터와 속성을 저장하는 변수입니다.

메서드 (Methods / 함수 / 행동):

객체가 수행할 수 있는 동작이나 기능을 정의하는 함수입니다. 주로 필드 데이터를 변경하거나 활용합니다.

생성자 (Constructor):

객체가 처음 만들어질 때(인스턴스화될 때) 호출되는 특별한 메서드로, 주로 필드 값을 초기화하는 역할을 합니다.
