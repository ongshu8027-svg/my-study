Java의 특징
1. 객체지향언어이다
클래스와 객체를 사용하여 프로그램을 구성한다.
2.독립성
JVM만 있으면 운영체제에 상관없이 살행 가능하다.
3.자동 메모리관리가 가능하다.
가비지 컬렉터가 사용하지 않는 메모리를 자동 정리한다.
4.보안성이 높다.
메모리를 직접 접근을 제한하여 안정성이 높다.
5.멀티스레드 지원
여러 작업을 동시에 처리 가능하다.

Java의 장점
운영체제 관계없이 실행가능하다
객체지향 프로그래밍 지원한다
보안성이 높다
다양한 라이브러리와 프레임워크 제공한다
유지보수가 용이하다

Java의 단점
실행속도가 C/C++보다 느리다.
메모리 사용양이 비교적 많다.
코드가 길고 복잡하다
